On Reporting Period

CY: Calendar Year A 12 months period always start from January and end in December

CCY: Current Calendar Year Dynamically using today's date to find out the Current Calendar Year : CY{{ now.year }}

FY: Financial Year A 12 months period as reckoned for taxing or accounting purpose

FY01M: First Financial Year Month The first month of TLMA's Financial Year is {{ tlma.fy1m | mthname }} (followed by FY02M: {{ (tlma.fy1m + 1) | mthname }} etc.)

CFY: Current Financial Year Dynamically using today's date({{ now | dAU }}) to find out the Current Financial Year : FY{{ tlma.current_fy }}
